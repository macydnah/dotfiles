# Changelog
All notable changes to this dotfiles repository are documented in this file.

## [2026-03-27]

### Added
- darkman light/dark mode switcher configuration.
- Neovim `basedpyright` LSP, replacing `jedi`.
- Tridactyl spotlight-style theme.
- OpenCode configuration with the `commit-review` skill.

### Changed
- Neovim Java workflow with `organizeImports` on save, plus `RealpathURI` and titlestring cleanup.
- Hyprland animations, input, window rules, idle/sleep behavior, battery-focused tuning, and Firefox-related rules.
- Waybar and Rofi visual/layout refinements.
- Foot terminal config updated for v1.21+ color syntax.
- Zathura scroll/recolor behavior and smooth-scroll bindings.
- Yazi plugin revisions and Vim plugin/source maintenance updates.
- Xresources DPI/font settings and shell network interface naming updates.
